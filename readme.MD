# CRUD with REST

Basic CRUD operations to learn rest APIs concepts!

## Tech Stack

**Client:** HTML, CSS, JS

**Server:** Node, Express, EJS

**Packages:** Faker, methodOverride

## Run Locally

Clone the project

```bash
  git clone https://link-to-project
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
